### ■Chapter09(2026.2.27)
Google Books APIの動作はAPIキーを指定した方が安定します
(指定しないと、ステータス429が返ってくることがある)。
安定的に開発を進めるために、APIキーを取得しましょう。
本書では、実装を簡易化するために *APIキー* 方式をお勧めします。
APIキーの作成方法は、
- https://docs.cloud.google.com/docs/authentication/api-keys?hl=ja

などをご参照下さい。必要なのは、以下の3点です
- プロジェクトの作成
- 認証情報　→　「＋認証情報の作成」を実施する
- 有効なAPIとサービス　→　「＋APIとサービスを有効にする」　→　Books APIを検索、「製品の詳細」で有効にするボタンを押す


ネットを検索するとスクショ付きのサイトあります。筆者のblogなど、合わせてご参照ください。

- https://genz0.blogspot.com/2026/02/othergoogle-books-apis.html
